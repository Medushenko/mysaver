# 🎯 1. Общее описание и цели проекта MySaver

## 📌 Миссия
Создать интуитивный, кроссплатформенный «швейцарский нож» для управления облачными хранилищами, который устраняет фрагментацию данных, автоматизирует рутинные файлоперации и даёт пользователю полный контроль через простой интерфейс, голос или чат-бота.

## 👥 Целевая аудитория
| Сегмент | Потребность | Фокус MVP |
|---------|-------------|-----------|
| Линейный пользователь | Копирование/перенос между облаками, управление со смартфона, понятные отчёты | ✅ Основной фокус |
| Продвинутый пользователь | Гибкие правила, квоты, расписание, API-интеграции | ⚡ Базовая поддержка |
| Малый бизнес / команды | Разделение доступа, аудит, экспорт отчётов | 🔵 Фаза 2+ |

## 🔑 Решаемые проблемы
1. **Фрагментация**: данные разбросаны по 3–5 облакам, нет единой точки управления.
2. **Ручная руина**: скачивание → загрузка при переносе между провайдерами.
3. **Отсутствие контроля**: непонятно, что скопировалось, где ошибка, как повторить.
4. **Мобильный дефицит**: большинство инструментов заточены под десктоп/CLI.

## 🧭 Ключевые принципы
- **Link & Done**: вставил ссылку/текст → система распознала → 1 клик «Старт».
- **Mobile-First**: полный функционал доступен со смартфона через PWA и Telegram-бот.
- **Transparent Execution**: реальный прогресс, пост-верификация, отчёт с кнопкой «Повторить ошибки».
- **Set & Forget**: фоновое выполнение + push-уведомления о ключевых событиях.
- **Progressive Disclosure**: простой интерфейс по умолчанию, расширенные настройки — по запросу.

## 📦 Границы MVP (Фаза 1)
- ✅ Парсинг ссылок из текста/сообщений
- ✅ Загрузка задач в форматах TXT/MD/YAML/JSON с интерактивным превью-деревом
- ✅ Копирование/перенос между Яндекс.Диск, Google Drive (S3/Dropbox — Фаза 2)
- ✅ Базовая верификация (`--checksum` в rclone) + отчёт с retry
- ✅ Мягкий контроль квот + fallback-уведомление
- ✅ Гостевой режим (сессия → авто-удаление токенов) + регистрация (опционально)
- ❌ Шифрование, сложные RBAC, десктопный mount, AI-поиск по контенту — отложены

## 🌍 Долгосрочная траектория
- Смарт-папки, система тегов, кросс-облачный поиск по метаданным
- Голосовое управление, ИИ-интерпретация свободных команд
- Десктопный агент (macOS/Windows/Linux) с on-demand загрузкой
- Enterprise-роли, аудит, биллинг, партнёрская экосистема
